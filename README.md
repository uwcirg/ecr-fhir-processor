# ecr-fhir-processor
Processes FHIR R4 resources used for chronic disease surveillance. 
This is for three CMS quality measures: 
1. CMS122 - Diabetes HbA1c Poor Control (≥9%)
2. CMS165 - Controlling High Blood Pressure
3. CMS2 - Depression Screening

Originally developed for Washington State Department of Health's TRAX "Translational Repository and Analytics Exchange" project.
